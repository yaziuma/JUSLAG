# 売買方向反転検証レポート（126営業日版）

生成日: 2026-06-02

## 1. 目的

シグナル上位を LONG にする現行方向が逆なのかを、126営業日のバックフィルで検証する。
`signal × 翌日 OC リターン` の符号整合性、四半期別の局面依存性を合わせて分析し、
「方向逆の疑いあり / なし / 保留」を判定する。

### 現行方向
signal 上位 q 分位 かつ signal ≥ 0.10 → **LONG**  
signal 下位 q 分位 かつ signal ≤ −0.10 → **SHORT**

### 反転方向
候補銘柄・閾値条件は現行と同一。ポジション方向のみ逆転。  
signal 上位 q 分位 かつ signal ≥ 0.10 → **SHORT**  
signal 下位 q 分位 かつ signal ≤ −0.10 → **LONG**

## 7. 所見

- 累積損益: 現行方向 -7.69% vs 反転方向 +10.98% （差 +18.67%pt、**反転優位**）。
- 採用日平均損益: 現行 -0.116% vs 反転 +0.171% （差 +0.287%pt）。
- `signal × next_oc` 全体平均: **-0.000045** → 中立。
- 四半期別では 1/3 四半期で反転優位（月別: 3/7 ヶ月）。

#### LONG / SHORT 側の主因
- 反転時の LONG 寄与: -0.095% （現行 SHORT 寄与の反転: -0.144% 相当）→ 改善。
- 反転時の SHORT 寄与: +0.266% （現行 LONG 寄与の反転: +0.260% 相当）→ 改善。

## 8. 結論

### 判定: **保留**

四半期別で優劣が混在しており、恒常的な方向逆とは断定できない。局面依存（特定レジームでのみ方向が逆転）の可能性がある。
