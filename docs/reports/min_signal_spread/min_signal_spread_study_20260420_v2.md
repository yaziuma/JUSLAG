# min_signal_spread 調査レポート v2（損益付き閾値感度分析版）

## 1. 目的

v1レポートでは `trade_signal_strength` の分布のみを分析したが、
本レポート（v2）では各閾値を適用したときの**実際の損益（gross P&L）**を
バックフィル計算し、閾値ごとのリターン・リスク特性を定量的に比較する。
