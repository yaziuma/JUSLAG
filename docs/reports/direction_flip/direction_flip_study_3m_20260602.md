# 売買方向反転検証レポート（63営業日版）

生成日: 2026-06-02

## 1. 目的

シグナル上位を LONG にする現行方向が逆なのかを、63営業日のバックフィルで検証する。
`signal × 翌日 OC リターン` の符号整合性、四半期別の局面依存性を合わせて分析し、
「方向逆の疑いあり / なし / 保留」を判定する。

### 現行方向
signal 上位 q 分位 かつ signal ≥ 0.10 → **LONG**  
signal 下位 q 分位 かつ signal ≤ −0.10 → **SHORT**

### 反転方向
候補銘柄・閾値条件は現行と同一。ポジション方向のみ逆転。  
signal 上位 q 分位 かつ signal ≥ 0.10 → **SHORT**  
signal 下位 q 分位 かつ signal ≤ −0.10 → **LONG**

## 7. 所見

- 累積損益: 現行方向 -14.40% vs 反転方向 +22.10% （差 +36.50%pt、**反転優位**）。
- 採用日平均損益: 現行 -0.479% vs 反転 +0.653% （差 +1.132%pt）。
- `signal × next_oc` 全体平均: **-0.000143** → 中立。
- 四半期別では 2/2 四半期で反転優位（月別: 3/4 ヶ月）。

#### LONG / SHORT 側の主因
- 反転時の LONG 寄与: +0.039% （現行 SHORT 寄与の反転: -0.067% 相当）→ 改善。
- 反転時の SHORT 寄与: +0.614% （現行 LONG 寄与の反転: +0.546% 相当）→ 改善。

## 8. 結論

### 判定: **方向逆の疑いあり（弱）**

反転方向の累積・四半期優位は確認できるが、`signal × next_oc` は中立。局面依存（レジームチェンジ）の可能性も残る。
