# min_signal_spread ポジションフィルタ比較レポート

## 1. 目的

min_signal_spread による日次フィルタ（閾値）と、min_long_signal / max_short_signal によるポジション件数可変化フィルタを組み合わせた場合の効果を比較検証する。
固定件数（ベースライン）との損益・リスク特性の差を定量化し、本番設定の候補を選定する。

## 2. 分析条件

| 項目 | 値 |
| --- | --- |
| 集計期間 | 2026-03-06 〜 2026-04-17 |
| 集計日数（信号生成済み） | 29 日 |
| 分析対象日数 (--days) | 30 日 |
| quantile_q | 0.3（上位/下位 30% を LONG/SHORT 対象） |
| 損益計算 | `compute_long_short_split()` による手動再現（1銘柄あたり等ウエイト） |
| 採用条件 | trade_signal_strength ≥ min_signal_spread **かつ** LONG件数 ≥ 1 |
